# 그린마린 양하 검수앱 (모바일용)

평택항 양하 검수 - Firebase 실시간 동기화

## 주요 기능
- BAPLIE EDI / 양하 리스트 / X-RAY 업로드
- 베이 플랜 (시프팅 자동 감지)
- 검수원 선택/교대 (이름별 작업 기록)
- Firebase 실시간 동기화 ☁

## 설치
```bash
npm install
npm run dev
npm run build
```

## 데이터 동기화
- 모든 검수원이 같은 항차 데이터 공유
- 완료 처리 시 즉시 Firebase 저장
- 수석검수 통계앱에서 실시간 확인
